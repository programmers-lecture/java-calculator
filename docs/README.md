### 요구사항 분석
- [x] 입력을 받는다.
- [x] 입력을 빈칸을 기준으로 숫자와 사칙연산을 분리한다.
  - [ ] 숫자인지 검증
  - [ ] 사칙연산이 아닌 다른 문자인지 검증
- [x] 연산을 한다.
- [ ] 결과값을 출력한다.

### 역할과 책임
* view
  * InputView 
    * 사용자로부터 입력을 받음
  * OutputView
    * 결과값 출력
* Calculate
  * 숫자가 맞는지 검증
  * 사칙연산이 맞는지 검증
* Operator
  * 사칙연산을 한다
* utils
  * StringParser
    * 빈칸을 기준으로 숫자,사칙연산,숫자를 분리한다.
