## 깃허브

**[java-calculator](https://github.com/programmers-lecture/java-calculator)**

## 요구사항
* [x] 입력을 받았을 경우 사칙연산(+, -, *, /)이 가능해야 한다.  
* [x] 피연산자(숫자)의 범위는 Integer로 한정한다.  
* [x] 숫자에 소수를 적용해본다.  
* [x] Operator를 Enum으로 분리하여 관리한다.  
* [ ] 사칙연산 우선순위를 적용한다.  
    - *, /와 +, -가 있다면 *, / 가 먼저 진행되어야 한다.

## 프로그램 실행 결과

```jsx
1 + 2
결과 : 3

1 - 2 + 5
결과 : 4

3 * 4
결과 : 12

3 + 2 * 3
결과 : 9
```
